# CUDA-projekt-2026
openCV's cv::equalizeHist function in CUDA C++
---

## Pipeline description:

- examples: contains python script that activates equalizeHist.cu to compute the cv::equalizeHist algorythm on GPU
- kernels: contains equalizeHist.cu kernel
- benchmark: contains benchmarks for GPU vs CPU execution of cv::equalizeHist algorythm on randomly generated image of fixed size
- setup.py: config file
- artifacts: images
